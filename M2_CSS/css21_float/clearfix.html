Css clearfix

Khi 1 thẻ html cao hơn thẻ mẹ chứa nó và nó được float left hoặc right
=> thì nó sẽ bị trượt ra khỏi thẻ
=> thẻ mẹ không còn chứa được thẻ con
vậy để giải quyết vấn đề này có 2 cách phổ biến
cách 1 : đặt thẻ mẹ thuộc tính overflow: hidden(cách này có nhược điểm)
cách 2: đó là sử dụng class clearfix và đặt class clearfix này cho thẻ mẹ .clearfix{
overflow:auto;
}
.clearfix::after{
content:"";
clear: both;
display:table;
}
